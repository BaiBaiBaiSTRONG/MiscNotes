## Wox 使用说明

+ Google Translate ：

    ```
    tr en/zh en/zh <words>
    ```

+ IP Address：

    ```
    ipadr
    ```

+ Remove USB:

    ```
    reu
    ```

+ Currency Converter:

    ```
    cc <number> cny usd 
    ```

+ Plugin.Github

    ```
    gh <repos>
    ```

+ 进制转换

    ```
    使用进制关键字+“~”开始。
    例：x~f //对16进制的 f 进行转换。
    例：b~1001 //对2进制的 1001 进行转换。
    关键字：x代表16进制，d代表10进制，o代表8进制，b代表2进制。
    ```

+ Weather

    ```
    forecast <Cityname >
    ```
